# Elasticsearch

## Version Mapping

| Chart | Elasticsearch |
| ----- | ------------- |
| 0.1.y | 5.6           |
| 1.x.y | 6.x           |
| 2.x.y | 7.x           |

For usage of Elasticsearch version > 6.8 or 7.1, I suggest to use official [ECK helm charts](https://www.elastic.co/guide/en/cloud-on-k8s/current/k8s-install-helm.html) to setup the whole cluster.
